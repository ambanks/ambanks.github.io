How to change Images

Welcome Section

  1. Go to the css Folder
  2. Open style.css
  3. Find section "welcome"
  4. Change the Background-Image "../img/welcome-bg.jpg"
  5. Save the file after changing

Work Gallery

Thumbnails

  1. Open index.html
  2. Find section "work"
  3. Go to the image you want to change
  4. Find the Thumbnail-Image source e.g. "img/gallery/thumbs/1.jpg"
  5. Change your Image
  6. Save the file after changing

Bigger Images

  1. Open index.html
  2. Find section "work"
  3. Go to the image you want to change
  4. Find the data-largesrc Attribute of your Image e.g. "data-largesrc="img/gallery/1.jpg"
  5. Change your Image
  6. Save the file after changing

Plugins

Change ScrollTo Plugin Options

  1. Go to the js Folder
  2. Open plugins.js
  3. Find "ScrollTo Navigation"
  4. Change e.g. scrollSpeed, add new scrollButtons
  5. Save the file after changing

Change Responsive Nav Plugin Options

  1. Open index.html
  2. Find "responsiveNav" (Bottom Scripts)
  3. Change e.g. transition speed
  4. Save the file after changing

Team Section

Change skill range

  1. Open index.html
  2. Find section "team"
  3. Go to the skills div
  4. Find the data-width attribute e.g. "data-width="85"" (=85/100%)
  5. Change the number (0-100) to get the percentage
  6. Save the file after changing

Work Gallery

Change gallery item data

  1. Open index.html
  2. Find section "work"
  3. Go to the image you want to change
  4. Find the data-title attribute e.g. "data-title="GT40""
  5. Change the title
  6. Save the file after changing

Change gallery item description

  1. Open index.html
  2. Find section "work"
  3. Go to the image you want to change
  4. Find the data-description attribute e.g. "data-description="Actually DIY VHS chillwave art party organic. Blog Brooklyn twee, Wes Anderson viral synth kogi next level butcher yr fixie umami lomo before they sold out.""
  5. Change the description
  6. Save the file after changing

Change filter & filter items

  1. Open index.html
  2. Find section "work"
  3. Go to the "gridfilter" ul
  4. Find the data-filter attribute e.g. "data-filter="all""
  5. Change the filter name
  6. Find ID "Grid"
  7. Every Grid list item must have the class "mix" and a filter class name e.g. li class="mix webdesign". Items with the same filter class will be filtered together
  8. Set you data filter and item class and save your file after changing
